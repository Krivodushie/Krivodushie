## ▸ Current Project - `The Glyph`

> *A persistent multiplayer pixel game. Roguelike progression. MMORPG-scale systems. Social layers. Built to last.*

**Co-developed with [Psiiiiiii](https://github.com/Psiiiiiii) - she owns the world you see, I own the world underneath.**

```
┌─────────────────────────────────────────────────────────┐
│  ARCHITECTURE OVERVIEW                                  │
├─────────────┬───────────────────────────────────────────┤
│  Transport  │  UDP (gameplay) + WebSockets (events)     │
│  Backend    │  Service-based microservice architecture  │
│  Messaging  │  Event-driven with async queues           │
│  State      │  Persistent world, PostgreSQL-first       │
│  Scale      │  Horizontally scalable infrastructure     │
└─────────────┴───────────────────────────────────────────┘
```

What I work on:
- **Game server** - authoritative backend, state sync, anti-cheat logic
- **Real-time layer** - UDP for game client, WS for chat/events
- **World persistence** - saving everything, losing nothing
- **Service mesh** - keeping microservices honest with each other
- **Gameplay systems** - progression mechanics, loot tables, world logic
